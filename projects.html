NGettext

A cross-platform .NET implementation of the GNU/Gettext library.

This fully managed library works fine on Microsoft .NET Framework version 2.0 or higher, Mono and .NET Core even on full-AOT runtimes. It is fully COM and CLS compatible.

GitHub Page

NLibuv

Libuv bindings for .NET

Provides the libuv API on .NET platform, as simple as possible. NLibuv takes care of platform invoke, marshaling, error handling and memory management, allowing you to consume libuv API as it is, safely on any platform supported by .NET.

GitHub Page

BaldurToolkit

Toolkit for creating .NET server applications.

Originally created as a core of the Baldur MMO game server engine, now available as a set of independent base components.

GitHub Page

Baldur Server Engine

Distributed and highly scalable MMO game server engine.

Written completely in C#. Cloud native (with Kubernetes support), cross-platform (.NET Core, .NET Framework, Mono).

Contains a high-performance parallel network stack, powerful entity system, self-healing infrastructure configuration, convenient DBMS-agnostic data storage layer, monitoring and logging systems, and a lot more options allowing you to create a stable and extensible game backend.
